# Movie Night Decision Dashboard

SQL + Power BI analysis of an IMDB dataset (~5000 movies) designed to help users quickly find a movie for a movie night.

## Tools Used

- SQL Server
- T-SQL
- Power BI (dashboard in progress)

## Project Goals

• Explore patterns in movie ratings and genres  
• Identify highly rated but lesser-known movies ("hidden gems")  
• Prepare data for an interactive movie discovery dashboard
## Dataset
~5000 movies including:
- title
- year
- rating
- votes
- runtime
- genres
- directors

## SQL Techniques Used
- Data cleaning
- STRING_SPLIT for genre normalization
- JOIN between movies and genre tables
- Aggregate functions
- Window functions (ROW_NUMBER)
- Exploratory data analysis

## Next Steps
Create a Power BI dashboard for interactive movie exploration.
